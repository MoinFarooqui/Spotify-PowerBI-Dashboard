## Project Overview

The Spotify Power BI Dashboard is an interactive business intelligence solution developed using Microsoft Power BI. The dashboard transforms Spotify Top 50 World data into meaningful visual insights, enabling users to analyze song rankings, artist performance, popularity trends, album distribution, release history, explicit content, and listening patterns through interactive reports and KPI-driven analytics.

The project demonstrates end-to-end dashboard development, including data preparation, DAX calculations, KPI creation, data modeling, and interactive business reporting.

---

## Business Objective

The objective of this project is to analyze Spotify's Top 50 World songs and provide meaningful insights into artist performance, song popularity, album characteristics, release trends, and listening patterns. The dashboard supports data-driven decision-making through interactive visualizations and business-focused KPIs.

---

## Dataset

The project uses the Spotify Top 50 World dataset containing song rankings, artist names, popularity scores, song duration, album types, release dates, track counts, explicit content indicators, and album artwork information.

---

## Tools & Technologies

Microsoft Power BI was used for data transformation, data modeling, DAX calculations, KPI development, dashboard creation, and interactive visualization.
